## 💻 Sobre o projeto

Gwenilib é uma biblioteca fictícia, com funcionalidades para cadastro de livros e usuarios, emprestimo de livros, além de solicitação de emprestimo.

---

## ⚙️ Funcionalidades

- [x] Cadastro/atualização de livros;
- [x] Cadastro de usuarios;
- [x] Listagem de livros disponíveis para emprestimo;
- [x] Solicitação de emprestimo;
- [x] Multa;

---

## 🛠 Tecnologias

As seguintes tecnologias foram utilizadas no desenvolvimento da API Rest do projeto:

- **Java 8**
- **Maven**
- **Postgres**
- **Hibernate**
